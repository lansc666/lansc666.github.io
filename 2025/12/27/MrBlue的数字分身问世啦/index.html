🤖 MrBlue 能聊什么？

这不是一个通用的 LLM，而是一个读过我所有文章的 AI。它的大脑里装载了我的两份核心记忆：

👉 点击与 AI 对话

🏫 环球高校足迹 (Campus Tour)

我已经完成了全球 50所高校 的环游打卡行动（从北工大到北大）。

  • 你可以问：“博主觉得牛津大学的氛围怎么样？”
  • 你可以问：“北师大珠海校区和海淀校区有什么区别？”
  • 你可以问：“哪所学校校内竟然有麦当劳？”（提示：北邮）
  • 它会调取我的真实日记，告诉你我的 私人体验、校训印象 以及那些有趣的 探校细节。

🏔️ 硬核地理科普 (Geo-Science)

基于公众号 “MrBlue的地理站” 的深度长文。

  • 你可以问：“为什么说内蒙古不只有草原？”（它会用经度地带性给你解释森林与荒漠）
  • 你可以问：“去墨脱的路为什么那么难修？”（它会告诉你扎墨公路背后的地质挑战）
  • 你可以问：“新疆天山有什么地理特色？”
  • 它会用严谨的地理视角，配合我文章中的通俗语言，为你解读祖国山河。

⚡ 幕后技术架构 (Tech Stack)

为了实现这个 AI 的合规访问与高可用性，我搭建了一套跨越 北京 -> 香港 -> 新加坡 -> 韩国 的网络链路。

数据链路可视化：
User ➔ Azure (Korea) ➔ [FRP 加密隧道] ➔ Huawei Cloud (Beijing) ➔ AI Inference

  • 流量入口 (Gateway)：利用 Azure (Korea) 的境外节点作为前置网关，通过 Nginx 进行流量分发与 SSL 加密。
  • 计算核心 (Backend)：利用 华为云 (Beijing) 运行 Dify 容器集群，通过 Swap 技术突破 2G 内存限制。
  • 推理引擎 (Model)：接入 MiMo-V2-Flash (LLM) 与 BGE-M3 (Embedding)，实现高精度的 RAG 检索。
  • 内网穿透 (Tunnel)：使用 FRP 协议打通 Azure 与华为云之间的加密隧道，隐藏源站 IP，保障安全。

欢迎大家点击体验！如果它说错了什么，那是 AI 产生了幻觉，或者是我的日记写得太潦草 😉。
