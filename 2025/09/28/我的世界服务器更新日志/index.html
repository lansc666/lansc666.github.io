我的世界服务器更新日志

发表于 2025-09-28 更新于 2026-03-23

【更新日志定期上传中】
感谢大家对 BNUESA MC 的支持！
QQ群（522811139）内现有一个 1.21.8 空岛 + 原版生存 服务器，允许离线登录，可以带自己的朋友一起玩~

基本信息

  • IP：play.simpfun.cn:14163
  • 版本：原版 1.21.8，无需任何 Mod 和数据包（理论上1.9.x-1.21.x均可以连入，推荐1.21.8）
  • 启动器：可使用 官方启动器、HMCL、PCL 等第三方启动器
  • 语音软件：Kook 👉 点击加入
  • 服务器出现任何问题可在群内反馈！

更新记录

v3.0

  • 支持1.9.x-1.21.x版本连入服务器（推荐使用最新稳定版1.21.8）
  • 加入创造世界，欢迎来超平坦世界上展现你的建筑技术！支持在游戏大厅通过传送门切换世界
  • 其他游戏体验上的优化

v2.2

  • 正式开放【原版生存】传送门，可从主世界出生点直达原版生存世界
  • 实现背包完全分离，不同世界互不影响
  • 简化并汉化聊天框的世界信息标注

v2.0

  • 加入原版生存世界，并支持用指令在各世界中切换
  • 取消对挂机时间的检测
  • 优化注册和登陆的时限与 IP 检测逻辑

v1.5

  • 空岛支持自主修改生物群系
  • 添加皮肤插件，优化游戏体验

v1.4

  • 新增空岛小队银行、跨世界物品清空和经验分离
  • 部分内容汉化，修复了一些 Bug

v1.3

  • 增加登陆功能，保障安全性
  • 添加箱子交易插件，即将进行测试
  • 定期清理掉落物，保障服务器性能
  • 其他提升游戏体验的功能

v1.2

  • 优化空岛生存体验
  • 新增任务系统、等级系统、访问功能、综合面板等
